Programming

Below are a few of the projects I've built or am currently working on. The projects with green headers contain links to pages with demos or more information about the projects.

Culver Current: Public Art Installation

A public art installation that opened on December 19th in front of Culver City Hall and will remain under active development throughout the entirety of 2020. For the project, I developed C# scripts which trigger Unity animations based upon real-time Twitter and Instagram activity related to Culver City. Click to view the official site for the installation.

Tools: C#, Unity, Selenium Webdriver

Drumify: A musical iOS Application

An iOS mobile app that helps users create drum beats from sounds they record. Click to view the demo video and website that I created for the app.

Tools: Swift, AudioKit, Digital Signal Processing

Forensic Evidence Web Platform

A cloud-based web application for the analysis and storage of criminal case information and digital evidence. Built during my time as an intern at the California Cybersecurity Complex. I created an extensive REST API using Flask, and hosted this API on an AWS EC2 instance to handle communication and data storage from the frontend client.

Tools: Python, Flask, AWS, PostgreSQL, Angular4

WeatherTrip: React Web Application

A React web app which provides a weather overview of relevant points along the route of a road trip. Supported by an Express NodeJS backend which processes data from the TomTom and Darksky APIs. The backend uses these APIs to indentify the most significant cities along a trip provided by the user and retrieves weather data and other information about these cities. The project is hosted on a Heroku server in the cloud.

Tools: React, Nodejs, Express, Heroku

Rock Band Website: badactormusic.com

A website for my rock band badactor. The site contains a landing page with relevant links to streaming platforms and social media, and a player page where the band's first EP can be listened to and lyrics can be viewed. I also used Javascript to create gradient and shadow animations to simulate a moving light source. Click to view the website.

Tools: HTML, CSS, Javascript, Bootstrap, jQuery

Handmade Community Helper App

An iOS mobile application which serves as an extension for the Amazon seller app. Constructed in close collaboration with Amazon employees over the course of the year for my Software Engineering Capstone project.

Tools: Swift, AWS, Nodejs, MongoDB, Agile Software Development

Stampede: A Multiplayer Unity Game

A top-down Unity video game which allows players to battle by influencing herds of buffalo to create stampedes. Includes music composed by me which dynamically changes to fit the action of the game.

Tools: C#, Unity, WWise, Boids pathing algorithm

Progressions Mobile App

An iOS application which helps the user create musical chord progressions by recommending common sequences of chords found in popular songs. Used Python to analyze a dataset of popular songs and create Markov Chains to represent common progressions.

Tools: Swift, Firebase, Python, Music21
